スクールTVの使い方

いつの間にか学習習慣が身につく新感覚の学習スタイルが登場!

  • パソコン・タブレット・スマホでいつでも見れる

  • 映像と連動したドリルで楽しく学べる

    • 独自の動画構成で飽きずに学習できる

「勉強が好きになれない」「苦手だからやりたくない」そんな声を受けて登場したのが、スクールTVです。

動画構成について

スクールTVの学習動画は、最初に再生する1本のみ2部構成になっています。


  • いきなりお勉強に集中するのは難しい。

    だからこそ、まずは頭をやわらかくするためにAL動画から始まります。

    「準備体操みたいだね！」

  • AL動画は最初の1本だけ再生されます。

    学習準備ができた2本目の動画からは再生されないのでシッカリお勉強に集中！

    「がんばるゾ～！」

  • AL動画では、不思議な食べ物や動物、世界の偉人、UFO…など、毎回さまざまなお話が登場します。

    「どんなお話があるかは見てからのお楽しみ★」


動画ドリルで腕試し！

動画とドリルが連動しているドリルでは、学習中に講師のお兄さんお姉さんが出した質問を実際に答えて行く事ができます。

  • 「時間内に答えられるかな～？」

    正解数とタイムでランキングが変わるので、ランキング上位を目指してがんばろう！

ラビボタンを使ってみよう！

動画の右側（もしくは下部）にあるのがラビボタンです。

動画を見ている途中、「わかった！」「むずかしい…」など、ボタンと同じ気持ちになったら押してみましょう。



  • 「レッスン中は何度でも押せるよ！」
